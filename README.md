# NAME

imaparchive.pl - Archive your IMAP emails into monthly archive folders

# SYNOPSIS

./imaparchive.pl

# DESCRIPTION

imaparchive.pl enumerates the emails in your INBOX and moves them into
monthly archive folders on your IMAP server.
